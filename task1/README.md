## Requirements

## Setup

The virtual environment is shared across both tasks and lives at the repo root.

**Create the environment (once):**
```bash
py -3.10 -m venv venv
```

**Activate:**
```bash
venv\Scripts\activate
```

**Install dependencies:**
```bash
pip install -r task1/requirements.txt
```
